School Performance Prediction



Tools Used

Jupyter Notebooks, Python, Microsoft Excel.


Predicting diabetic outcomes, this project employs a comprehensive machine learning approach. Exploratory Data Analysis (EDA) uncovers influential factors, Logistic Regression provides foundational insights, and Artificial Neural Networks (ANN) delve into intricate data patterns. K-Means clustering enables patient stratification, while Convolutional Neural Networks (CNN) process medical images, and Recurrent Neural Networks (RNN) model temporal dependencies in health records. This amalgamation aims to create a robust predictive model for diabetic outcomes, facilitating early intervention and personalized care.
